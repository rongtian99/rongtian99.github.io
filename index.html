【虚幻4C++类】GameModeBase类

索引

UE4官方文档->第一人称射击游戏教程->设置项目->游戏模式基础类

介绍

Game Mode 定义游戏的规则和胜利条件。Game Mode 还将设置用于部分基础游戏性框架类型（包括 Pawn、玩家控制器和 HUD）的默认类。
官方文档在该处编写了一个GameModeBase类用于在ue4关卡运行时在屏幕左上角显示相关信息(一种游戏模式,一种规则)

实现示例图

实现代码

#pragma once

#include "CoreMinimal.h"
#include "GameFramework/GameModeBase.h"
#include "FPSProjectGameModeBase.generated.h"

UCLASS()
class FPSPROJECT_API AFPSProjectGameModeBase : public AGameModeBase
{
    GENERATED_BODY()
        virtual void StartPlay() override{
        Super::StartPlay();      //子类的StartPlay函数继承父类该函数
//同时子类重的StartPlay函数要完成以下
    if (GEngine)  //若ue4开始运行时
    {
        //显示调试信息五秒
        //-1“键”值（首个参数）说明我们无需更新或刷新此消息
        GEngine->AddOnScreenDebugMessage(-1, 5.0f, FColor::Yellow, TEXT("Hello World, this is FPSGameModeBase!"));
    }
    }       //通过此函数声明覆盖StartPlay函数，游戏开始时便会将日志消息显示到屏幕上

};
